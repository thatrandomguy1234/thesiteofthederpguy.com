Official Instruct+ website. Instruct+ is an easy to use low level programming language that can't do much. If can just be used to connect to your operating system's terminal, make directories, remove directories and that's it. This is true for the Version 1 of Intruct+. Version 2 is still in the workings.
To get started.
Download the Instruct+ Compiler ->
Create a file. Open the file with VSCode and type "start{}end"
Between the curly brackets "{}", press enter and type in either of these keywords below with following text.

keywords

print
execute
delay
mkdir
remove dir
kill
pass
